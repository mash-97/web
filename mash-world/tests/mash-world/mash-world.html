Sturdy Bubble World

Rules

  • A new object is created if a randomly produced value (0-1) crosses the Object Creation Threshold at every Simulation Time Interval. It gets a random velocity/step vector (vx, vy) between the Minimum Speed and Maximum Speed, a random size between Minimum Size and Maximum Size (same dimension), and a random color from (red, green, blue) with the Default Lifespan.
  • An object will assimilate any other object with same color if the size difference between them crosses the Interaction Threshold.
  • For different colored objects, the bigger one will be disintegrated into several smaller objects inheriting the smaller object's size.
  • *Several other ingrained rules applied* to stabilize the simulation to some extent, and the simulation might act strange depending on the configuration edges.
  • The simulation is an approximation!
