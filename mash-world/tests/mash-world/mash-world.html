Mash World

Rules

  • A new object is created if a random value crosses the Object Creation Threshold. The newly created object gets a random velocity/step vector (vx, vy) between the Minimum Speed and Maximum Speed, a random size between Minimum Size and Maximum Size, and a random color from (red, green, blue).
  • Only an object with different color and comparably big size assimilates other object with different color and comparably small size. (e.g. a Red colored object with 10*10 sizes will devour a Blue colored object with 9*9 sizes).
  • Two same colored objects doesn't do anything if their sizes ratio difference doesn't crosses the Smashing Threshold. If otherwise the bigger object will be broken down into several pieces of objects with the size of the smaller object between them, resulting the different pieces moving into different directions.
  • Every object gets an initial Default Lifespan. An object gains the lifespan of others as according to the assimiliation rules.
